# SkillLearner

SkillLearner is a mentorship booking platform built with Django where students can connect with teachers and book mentorship sessions.

## Features

- Teacher & Student role system
- Teacher profile management
- Students can browse teachers
- Booking system
- Teachers approve/reject requests
- Students cancel pending bookings
- Role-based access control

## Tech Stack

Backend:
Python
Django
Django REST Framework

Frontend:
HTML
CSS
JavaScript

Database:
SQLite

## Run Project

Clone repository

git clone https://github.com/Raunaksharma0410/skill-learner.git

Install dependencies

pip install -r requirements.txt

Run server

python manage.py runserver
